# feminic-dio-
Numero de casos de feminicídio no Brasil

# 📖 Storytelling: O Retrato da Violência (2015-2025)

### 1. O Despertar de uma Década
Em 2015, o Brasil registrou **535 ocorrências** de feminicídio. O que deveria ser o marco para o controle do crime tornou-se o início de uma escalada alarmante. Validamos via **SQLite** que o país permitiu que esses números triplicassem em 10 anos.

### 2. O Ápice do Retrocesso
Os dados de 2025 revelam um **recorde histórico**: o Brasil atingiu a marca de **1.470 feminicídios**. 
* Aumento de **174,77%** em relação a 2015.
* Estagnação no topo: 2024 (**1.464**) vs 2025 (**1.470**).

### 3. A Anatomia do Recorde (Dados G1/Sinesp)
A análise em **Python** destaca os epicentros em 2025:
* **São Paulo**: 233 casos (Dados parciais).
* **Minas Gerais**: 139 casos.
* **Rio de Janeiro**: 104 casos.

### 4. Meta de Redução Estratégica
Para retornar ao nível de 2015, é necessário reduzir os casos em **63,6%** nacionalmente.
* **São Paulo** deve baixar para **85** casos.
* **Minas Gerais** para **51** casos.
* **Rio de Janeiro** para **38** casos.


> 
> **Nota Técnica:** Este estudo utilizou dados oficiais do **Ministério da Justiça**, processados via **SQLite** para validação lógica e apresentados via **Python** para análise estatística e visual.
